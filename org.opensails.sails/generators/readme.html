Thank you for taking the time to download this sails distribution. We hope you enjoy using it.

Steps for running this Sails Distribution from the file system.

  • Required: java 1.5 jdk must be on your path.
  • First extract the zip file to a destination of your choice.
  • Then run $application.name;.bat or $application.name;.sh file as appropriate for your OS.

Steps for running this Sails Distribution from Eclipse 3.

  • Import the Archive (File > Import... Existing Project, browse to the archive)
  • Required: java 1.5 jdk must be set as the jre for this project.
  • Then run the $application.name; launch config. (Run... $application.name;)
View your Sails Application "$application.name" running on http://localhost:1111
